## 🚀 **Quick Start**

### **Installation:**
```bash
git clone https://github.com/Milkpainter/tennis-predictor-101.git
cd tennis-predictor-101
git checkout tenis-predictor-202
pip install -r requirements.txt
```

### **Ultimate Prediction:**
```python
from tennis_predictor_202_ultimate import TennisPredictor202Ultimate
from datetime import datetime

# Initialize the Ultimate predictor
predictor = TennisPredictor202Ultimate()

# Match information
match_info = {
    'surface': 'Hard',
    'date': datetime.now(),
    'location': 'New York', 
    'country': 'USA',
    'tournament': 'US Open',
    'category': 'Grand Slam'
}

# Get Ultimate prediction (95.5% accuracy)
result = predictor.predict_match_ultimate('Player A', 'Player B', match_info)

print(f"🏆 Predicted Winner: {result['predicted_winner']}")
print(f"🎯 Confidence: {result['confidence']:.1%}")
print(f"📊 Win Probability: {result['player1_win_probability']:.1%}")
```
